Detailed Description

MainView class is practically "main body" of the app when Login class responsible for app working moves to it after successful login.

Tab widget of MainView got four pages with 4 classes added to it's content.
Pages of tab widget:
a) Read - Read class is responsible for it's content
b) Create - Create class is responsible for it's content
c) Update - Update class is responsible for it's content
d) Delete - Delete class is responsible for it's content
So this clas is responsible for working with instances of classes mentioned above and with opening and closing
of teachers database.
